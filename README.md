# StroiEkb — Производство МАФ и фасадных решений

Сайт производственной компании полного цикла. Мы создаем малые архитектурные формы (МАФ), парковую мебель, корзины для кондиционеров и вентиляционные системы.

## Характеристики проекта

- **Фреймворк**: Next.js 16 (App Router)
- **Технологии**: TypeScript, Tailwind CSS 4, Framer Motion 12
- **3D**: Интеграция Spline
- **База данных**: Supabase (сбор заявок)
- **Деплой**: Docker (standalone)

## Начало работы

Запустите сервер разработки:

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере, чтобы увидеть результат.
